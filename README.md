## Introduction

ABC Cinema has been a leader in the film industry with a strong reputation for excellent movie experiences. The current digital infrastructure lacked modern features such as online ticket booking, secure payments, and notifications. This project addresses these challenges by providing a reliable, user-friendly, and feature-rich online system for customers and administrators.

---

## Project Objectives

1. Streamlined ticket booking process.  
2. User authentication and role-based security.  
3. Real-time seat availability updates.  
4. Integration with PayPal for secure payments.  
5. Admin functionalities: manage events, tickets, refunds, and user queries.  
6. Notification system via email and SMS.  
7. Cross-platform compatibility (desktop and mobile).  
8. High-performance, error-free system.  
9. Feedback and support system for users.

---

## Features

- **Online Ticket Reservation**: Browse movies, select showtimes, choose seats.  
- **Payment Processing**: Secure PayPal integration with SSL encryption.  
- **Feedback Visualization**: Admin panel shows customer feedback trends for data-driven decisions.  
- **Notification System**: Email/SMS notifications for bookings and cancellations.  
- **Collision Avoidance**: Ensures seat booking integrity and prevents overbooking.  
- **Admin Panel**: Manage movies, showtimes, reservations, and user feedback.
